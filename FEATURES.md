# 功能列表

本项目旨在展示 Tauri 2.0 与 Next.js 15 的集成，特别是前端和 Rust 后端之间的通信机制。

## 基础功能

1. **基本命令调用**
   - 从前端调用简单的 Rust 函数
   - 展示使用 `invoke` API 调用 Rust 命令

2. **参数化命令调用**
   - 从前端传递参数给 Rust 函数
   - 接收结构化的返回数据

3. **事件系统**
   - 从 Rust 向前端发送全局事件
   - 从 Rust 向特定窗口发送事件
   - 在前端监听和处理事件

## 演示内容

1. **问候示例**
   - 简单的 `greet` 命令，返回带有时间戳的问候语

2. **计算器示例**
   - 演示如何传递参数给 Rust 函数
   - 展示如何处理复杂的结构化返回数据
   - 处理错误情况（例如除以零）

3. **进程监控模拟**
   - 从 Rust 后台任务发送定期更新
   - 在前端实时显示这些更新
   - 展示事件的注册和取消注册机制
